App.js — современный javascript-фреймворк для создания десктопоподобных приложений.

Включает в себя классы для создания окон, в том числе диалоговых, меню.

  • Window
  • Menu

Компоненты

  • Конструктор Interface.Window

    Оборачивает содержимое блока в окно, подключает для него js-модуль.

    var code = new Interface.Window({
        content_el : utils.dom.id('code'),
        overlay : true,
        movable : true,
        js_module : function(self) {
            return {
                print: function() {
                    self.el.innerHTML = selected_keys.generate_config();
                    self.show();
                }
            };
        }
    });

    Параметры:

    • content_el* : dom-элемент для вставки в окно,
    • overlay : добавить overlay,
    • movable: можно перетаскивать окно,
    • closable: можно закрывать,
    • layout: стиль окна,
    • js_module: функция, которая возвращает объект с методами, которые будут подмешаны в окно.

    Методы:

    • show — показывает окно,
    • hide — скрывает окно.

  • Меню

    Создает многоуровневое меню (до 4-х уровней), каждая ссылка - действие.
